# Sistema de Cadastro de Alunos em Python

Projeto simples feito em Python para praticar lógica de programação.

## Funcionalidades

- Cadastro de alunos
- Registro de notas
- Cálculo automático da média
- Situação do aluno (Aprovado / Recuperação / Reprovado)
- Listagem de alunos cadastrados

## Tecnologias

- Python 3

## Como executar

1. Clone o repositório
2. Execute:

python sistema.py
